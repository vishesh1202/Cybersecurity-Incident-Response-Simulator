# Cybersecurity Incident Response Simulator

A hands-on simulator replicating 8 real-world attack scenarios to practise SOC incident response workflows.

## What it does
- Simulates phishing, lateral movement, and ransomware deployment scenarios
- Guides the user through Detection → Analysis → Containment → Eradication → Recovery phases
- Reduced average simulated response time from 45 mins to 12 mins through playbook drills

## Tools & Technologies
Python | Custom playbook engine | Log simulation framework
